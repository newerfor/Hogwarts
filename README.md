Hogwarts Characters App 📱
🏰 Обзор приложения
Hogwarts Characters — это мобильное приложение, посвященное вселенной Гарри Поттера, которое предоставляет полную информацию о персонажах школы волшебства Хогвартс. Приложение заменяет бумажные справочники и онлайн-вики, предлагая удобный доступ к данным о персонажах прямо на вашем устройстве.

🎯 Основные функции
👥 Управление персонажами Хогвартса
Полные профили персонажей с фотографиями, датой рождения, факультетом и волшебными способностями

Разделение на краткую и детальную информацию — список персонажей и полные профили

Волшебные атрибуты — информация о патронусах, волшебных палочках, предках

Статусы персонажей — живой/умерший, студент/преподаватель, волшебник/магл

🏠 Факультетская система
Гриффиндор, Слизерин, Пуффендуй, Когтевран — полная информация по каждому факультету

Тематический дизайн — уникальные фоны и иконки для каждого факультета

Эмблемы и символика — мгновенное распознавание принадлежности персонажей

🛠 Технологический стек
Архитектура и паттерны
Clean Architecture с четким разделением на слои:

Data (Room, Retrofit, Mappers)

Domain (UseCases, Models, Repository Interfaces)

Presentation (ViewModels, Compose UI)

MVVM (Model-View-ViewModel) с StateFlow

Repository Pattern для абстракции источников данных

Dependency Injection через Koin

Room — локальная база данных с миграциями

Koin — легковесный DI контейнер
